# Body
The purpose of this part is to convey realistic alive-like movements to the MIST's body. Esia hasn't yet determined what the final motor choices and William haven't worked on the wiring yet so I have near-zero knowledge on how to control each motor before actually moving to the higher-level animations.

Last time I've seen the design of the robot's leg, it felt like we're only having brushless motors for legs, so I'm unsure if the legs will be articulated or we'll have car-like control. Esia stated that servo motors are straight and I must figure out how to simulate MIST's weight on the legs folding.

Maybe, following Disney's approach, we could create some pre-made animations in Blender and export them in some format for the robot?
